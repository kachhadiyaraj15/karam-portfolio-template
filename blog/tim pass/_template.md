## Supported syntax

This template supports the same syntax as [blog/_template.md](../_template.md), including:

- standard markdown headings, lists, links, tables, code, images
- task lists, footnotes, definition lists, highlight, math
- Mermaid diagrams
- Obsidian-style wiki links and embeds
- callouts, comments, tags, details blocks
- image/file embeds such as `![[image.png]]` and `![[file.pdf]]`

### Quick examples

